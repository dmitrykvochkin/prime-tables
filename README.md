# prime-tables
  - Generate primes multiplication table.

Developed by [Dmitry Kvochkin][df1]

### Running the Application
Clone the repo, then:
```sh
$ cd prime-tables
$ npm install
$ node index.js --n=10
```

### Testing
```sh
$ npm test
```
### Benchmarking
```sh
$ npm run benchmark
```
